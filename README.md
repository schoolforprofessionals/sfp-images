# sfp-images

`sfp-images` is a Bun and TypeScript package for dynamically generating branded images for School for Professionals sites.

## Overview

The package is intended to support:

- a shared internal image-generation core
- a CLI for scripted or repeatable image generation
- consistent output across School for Professionals sites

Exact CLI arguments and generation inputs will be defined as the implementation takes shape.
